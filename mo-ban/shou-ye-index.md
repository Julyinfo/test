# 首頁 Index

![](../.gitbook/assets/index_image_01.png)

首頁的固定順序如下：

* 第一區塊 - 將顯示頁首，並且固定於視窗上方。
* 第二區塊 - 將配置快速選單，使進入網站操作更加便捷快速。
* 第三區塊 - 安插服務狀態提醒使用者目前所提供之服務。
